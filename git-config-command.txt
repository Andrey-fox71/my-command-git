Git — это система контроля версий, 
которая помогает отслеживать изменения в проекте. Отслеживание изменени файла надо
инициализировать командой git add file.txt или всех файлов в папке командой git add --all
git add также надо применять для запоминания нового состояния файла, т.е. изминил файл и если всё ок
то add'нул его.

.gitcinfig хранит настройки и информацию
git config --global user.name "User-name" - установит имя пользователя
$ git config --global user.email username@yandex.ru - установит email 

cat ~/.gitconfig  - выведет имя пользователя и email
git config --list - выведет всю инфу из gitconfig'a
git init - инициализация проекта
git branch -m master main - переименовать ветку из master в main (важно)
git status - проверка состояния репозитория
git add file.txt - начать отслеживание изменений указанного файла(file.txt это пример названия)
git add --all - начать отслефивание всех файлов в папке
git add . - добавить к отслеживанию всю папку целиком(скорее всего +вложенные папки)


Инициализация локального места под проект(проекта):

Сначало создать(если не создана) и зайти в папку проекта.(пример: cd ~/dev/second-project)
Затем ввсти команду git init

Чтобы разгитить папку(находясь в ней) - rm -rf .git # удалит скрытую подпапку .git
