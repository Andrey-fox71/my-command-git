Список команд командной строки Git:

pwd - показывает где сей час находитесь (какая папка активна)
~ - так обозначается домашняя директрия
. - текущая директория
.. - родительская директория
cd (путь) - перейти в казанную директорию (если есть пробелы в названии папки, то необходимо использовать кавыки)
cd ~ - перейти в домашнюю директорию
cd .. - перейти на один уровень выше(назад)
cd . - для запуска скриптов и программ, которые принимают папку в качестве параметра(как пример)
ls - показывает файлы находящиеся в директории
ls -a - показывает файлы в директории, в том числе скрытые("." в начале названия)
ls ~ ; ls . ; ls .. - будут показывать содержимое этих директрий
touch - создать файл в текущей директории. Указывать имя и расширение файла.
mkdir - создать директорию в текущей директории
mkdir -p - создать структуру директорий(пример: dir1/dir2/dir3)
mkdir ~ ; mkdir .. - создать директории в указанных местах(в домашней и на один уровень выше)
cp что куда - команда копирования папок и файлов(можно: cp что что что куда)
mv что куда - команда перемещения папок и файлов(можно: mv что что что куда)
cat - чтение файлов в консоль (только текстовые файлы)
rm - удалить файл
rmdir - удалить директорию (удаляет только пустые)
rm -r - удалить папку со всем содержимым
&& - совмещает команды(пример: mkdir second-project && cd second-project && touch index.html style.css)


Примеры:

mkdir ~/my-git-projects - создаст папку my-git-projects внутри домашней директории.

touch ../../file.txt - создаст файл file.txt на две папки выше по иерархии.
 Допустим, если вы находитесь в директории projects/git/hello, 
команда touch ../../file.txt создаст файл по такому пути: projects/file.txt

cp index.html style.css script.js src/
скопировали три файла (index.html, style.css и script.js) в папку src
